# OmniSpecs Advisor

A simple Python tool that helps you look up tech specs fast. Instead of searching the web, just type the name of a phone or laptop, and the program instantly shows its RAM, storage, and screen size from a built-in list. It also gives you a "Quick Verdict" to tell you if the device is good for gaming, school work, or basic use.

## Features

- 🔍 Quick device lookup by name
- 📱 Built-in database (no internet required)
- ⚡ Instant "Quick Verdict" recommendations
- 🔤 Case-insensitive partial matching
